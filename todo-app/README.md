# 📱 Планер-Органайзер PWA

## Установка на телефон

### Android (Chrome)
1. Откройте сайт в браузере Chrome
2. Нажмите на меню (три точки)
3. Выберите **"Установить приложение"** или **"Добавить на главный экран"**
4. Подтвердите установку
5. Иконка появится на главном экране

### iPhone (Safari)
1. Откройте сайт в Safari
2. Нажмите кнопку **"Поделиться"** (квадрат со стрелкой)
3. Прокрутите вниз и выберите **"На экран «Домой»"**
4. Введите название (например, "Планер")
5. Нажмите **"Добавить"**
6. Иконка появится на главном экране

## Запуск локально

Для работы PWA нужен HTTPS или localhost. Есть несколько способов:

### Способ 1: Python (быстро)
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
Затем откройте `http://localhost:8000`

### Способ 2: Node.js
```bash
npx http-server -p 8000
```

### Способ 3: VS Code
Установите расширение **"Live Server"** и нажмите **"Go Live"**

### Способ 4: PHP
```bash
php -S localhost:8000
```

## Генерация иконок

1. Откройте `icons/generate-icons.html` в браузере
2. Браузер скачает все необходимые PNG иконки
3. Скопируйте их в папку `icons/`

Или используйте онлайн-генератор:
- [PWA Icon Generator](https://realfavicongenerator.net/)
- [App Icon Generator](https://appicon.co/)

## Возможности

✅ **Календарь** - визуальный календарь с задачами
✅ **Задачи** - добавление, редактирование, удаление
✅ **Категории** - работа, дом, учёба, личное, здоровье
✅ **Приоритеты** - низкий, средний, высокий, срочно
✅ **Напоминания** - уведомления о задачах
✅ **Заметки** - текстовые заметки с сохранением
✅ **Статистика** - прогресс выполнения задач
✅ **Офлайн режим** - работает без интернета
✅ **Мобильная версия** - адаптировано для телефонов

## Публикация в интернете

Для доступа с телефона опубликуйте приложение:

### Бесплатные хостинги:
- **GitHub Pages** - загрузите репозиторий на GitHub
- **Netlify** - перетащите папку в веб-интерфейс
- **Vercel** - подключите GitHub репозиторий
- **Cloudflare Pages** - быстрая публикация

### Пример для GitHub Pages:
1. Создайте репозиторий на GitHub
2. Загрузите файлы
3. В настройках включите GitHub Pages
4. Откройте ссылку с телефона

## Требования

- Современный браузер (Chrome 67+, Safari 11.3+, Firefox 68+)
- HTTPS соединение (или localhost)
- Включенный JavaScript
